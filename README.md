# Mitchelle Transportation LTD

![Status](https://img.shields.io/badge/Status-Live-brightgreen) ![AWS](https://img.shields.io/badge/Cloud-AWS%20DynamoDB-orange) ![React](https://img.shields.io/badge/Frontend-React.js-blue)

A cloud-powered bus ticketing and payment system for travelers across Nigeria. Book tickets, make payments, and track your bus in real time.

## Features
- Book bus tickets online in seconds
- Multiple payment options, Card, Bank Transfer, USSD, OPay, PalmPay, Pay on Arrival
- Real time bus tracking
- Admin dashboard with live analytics
- Cloud storage powered by AWS DynamoDB

## Tech Stack
React.js, Node.js, Express.js, AWS DynamoDB (us-west-2)

## Routes Available
Lagos to Abuja, Lagos to Port Harcourt, Abuja to Kano, Lagos to Ibadan, Abuja to Enugu, Port Harcourt to Calabar

## Setup
Clone the repo, add AWS credentials to `backend/.env`, run `node server.js` in backend and `npm start` in frontend.
